IF ELSE STATEMENT

if
- Executes code when condition is true.

if else
- Executes one block if true.
- Executes another block if false.

else if
- Used to check multiple conditions.

Syntax:

if(condition){
   code
}
else{
   code
}

Use Cases:
- Login System
- Age Verification
- Grade System
- Access Control

###################### Switch STATEMENT ################

Definition:
- Switch statement is used to execute one block of code from multiple options.

Syntax:

switch(expression){
    case value1:
        // code
        break;

    case value2:
        // code
        break;

    default:
        // code
}

Keywords:
switch  -> Starts the switch statement.
case    -> Checks a value.
break   -> Stops execution.
default -> Runs if no case matches.

Best Use:
- Day Finder
- Calculator
- ATM Menu
- Language Selection
- Menu System

Difference:

if else
- Used for conditions.

switch
- Used for fixed values.
