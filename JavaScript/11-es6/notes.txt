=====================================================
01. What is ES6?
=====================================================

Definition (English)
ES6 (ECMAScript 2015) is the 6th version of JavaScript. It introduced modern features that make code cleaner, shorter, and easier to maintain.

Definition (Hinglish)
ES6 JavaScript ka modern version hai jo 2015 me release hua. Isme naye features aaye jisse code readable, short aur powerful ho gaya.

Why do we use ES6?

• Clean Code
• Less Code
• Easy to Read
• Better Performance
• Modern JavaScript
• React & Node.js use ES6

Real Project Use

✔ React
✔ Next.js
✔ Node.js
✔ Express.js
✔ APIs

Interview Answer

Q. What is ES6?

Answer:
ES6 (ECMAScript 2015) is the modern version of JavaScript released in 2015. It introduced features like let, const, arrow functions, classes, promises, template literals and modules.

02. let
=====================================================

Definition (English)

let is used to declare variables whose value can be changed later.

Definition (Hinglish)

let se variable banate hain jiska value baad me change kar sakte hain.

Syntax

let variableName = value;

Example

let age = 20;

age = 21;

console.log(age);

Output

21

Explanation

Step 1

let age = 20;

Variable create hua.

Step 2

age = 21;

Value update ho gayi.

Step 3

console.log(age);

Output = 21

Real Project Use

✔ Counter
✔ Score
✔ User Input
✔ Shopping Cart

Advantages

• Value change ho sakti hai.
• Block Scope.
• Better than var.

Common Mistake

❌

let age = 20;

let age = 25;

Error

✅

let age = 20;

age = 25;

Interview Question

Q. Difference between var and let?

Answer

var

• Function Scope
• Re-declare Allowed
• Old JavaScript

let

• Block Scope
• Re-declare Not Allowed
• Modern JavaScript

06. Arrow Functions
=====================================================

Definition (Hinglish)

Arrow Function function likhne ka short aur modern tarika hai.

Syntax

};

20

Explanation

Step 1

const multiply = (a,b)=>{ }

Arrow Function create hua.

Step 2

return a*b

5 × 4

↓

✔ Event Handling

• Short Code
• Easy to Read
• Modern Syntax

a+b
}

Output

undefined

✅

return a+b;
}

const sum=(a,b)=>a+b;
