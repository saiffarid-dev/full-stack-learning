==========================================
03. document Object
==========================================

Definition (English)

The document object represents the entire HTML document. It is the main object of the DOM used to access and manipulate HTML elements.

------------------------------------------

Definition (Hinglish)

document object poori HTML page ko represent karta hai. JavaScript isi object ki help se HTML ko access aur modify karti hai.

------------------------------------------

Best Practices
